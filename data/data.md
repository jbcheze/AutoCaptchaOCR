# Dataset

The datasets used for training and evaluation are not included in this repository due to size limitations.

They can be downloaded from Google Drive:

https://drive.google.com/drive/folders/1GxwBTh20oTj5RF77SB6fchJqCECVv8DC?usp=sharing

---

## Installation

1. Download the dataset archive from the Google Drive link
2. Extract the files into this `data/` directory

Expected structure:

```
data/
├── raw/
├── processed/
└── samples/
```

---

## Notes

- The dataset contains CAPTCHA images used for OCR training
- Files are excluded from GitHub to keep the repository lightweight
- This setup follows standard machine learning reproducibility practices
